Welcome to WebEditRun - A Standalone Web App Editor


WebEditRun is a simple standalone web application which allows the editing and running of HTML/Javascript/CSS code in a standalone web page. WebEditRun is meant be used for testing ideas or prototyping.


Features

  • Syntax highlighting supported code editor based on the excellent ACE code editor
  • Live runnning of code in a separate frame
  • Run either top+bottom or side-by-side
  • Load test files on the fly via URL such as this: WebEditRun.html&src=myFileGoesHere
  • no server side report required
  • no CDN or web dependancies required so can run standalone via filesystem
  • Clone me on github: WebEditRun on Github
