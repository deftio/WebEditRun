Welcome to LiveEdit.html


LiveEdit is a simple example of a standalone html/javascript/css text editor web app that can be used for testing ideas or prototyping.

LiveEdit presents a simple text editor with syntax highlighting which can be edited live in the page. Then the HTML/Javascript/CSS code can be run by pressing the [Run Code!] button. At this point the code in the editor is loaded in to the frame on the same page and run. This can be repeated as many times as necessary.

Features

• Syntax highlighting supported code editor based on the excellent ACE code editor.
• Live runnning of code in a separate frame
• Load test files on the fly via URL such as this: liveedit.html&src=myFileGoesHere
• no server side report required
• no CDN or web dependancies required so can run standalone via filesystem
